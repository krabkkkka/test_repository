## Документация

### Описание тестов
- Тест проверки результатов поиска в Яндексе
- Тест проверки раздела Картинки в Яндексе

### Запуск тестов
```commandline
python -m pytest
```

### Получение отчета о прохождении тестов через Allure
```commandline
py.test --alluredir=allure_result_folder ./tests
```
